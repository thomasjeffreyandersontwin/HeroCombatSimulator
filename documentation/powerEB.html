Power Overview

Power Energy Blast

Description

Implements power energy blast.  D6, defense(PD/ED), and stun only are configurable.

Functional Overview

If configured for stun only, the damage effect is generated.  Otherwise, the generic damage generation from the battleEngine is used.
The Ability.CANSPREAD value/pair is set to True, to indicate that the EB can be spread.

Known Caveats

None.
