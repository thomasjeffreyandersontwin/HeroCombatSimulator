Power Overview

Power Aid

Description

Implements the Power Aid.  Currently only the simplified healing aid is implemented.  The number of dice of healing is configurable.

Functional Overview

When the power is triggered, the power generates an effect with limited stun and body aid of the appropriate amount.

Known Caveats

None.
