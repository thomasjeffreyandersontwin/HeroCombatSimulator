Effect Overview

Effect KnockedDown

Description

Applied to character in situations where the character is either knockdown or tumbling (in the air).

Functional Overview

Uses addDCVDefenseModifiers method to adjust DCV appropriately.
Uses addAction method to add a "Stand up" action to the special actions menu.

Known Caveats

None.
