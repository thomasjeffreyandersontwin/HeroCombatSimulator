Power Overview

Maneuver Grab

Description

Implements a standard grab maneuver.

Functional Overview

If the attack is successful, a Grabbed Effect is applied to the target.  If the effect is applied sucessfully (ie Target didn't break free with casual strength), Grab query the user for what the attacker is doing (squeezing, throwing, etc.) and processes an additional attack on the target, via the postTrigger method.  The postTrigger is necessary since Grab doesn't know during triggerPower() what is going to happen with the grab effect.
Two miniture powers exist, squeeze and throw, which grab activates appropriately, on the fly, to process those actions.
Once a character is grabbed, the source of the grabbed gets the Special Actions of throw, squeeze, and release.  These are populated by the Grabbing effect which is applied to the source.

Known Caveats

None currently known, but this is one of the more complicated powers, with lots of interactions.  Some conflicts are expected.
