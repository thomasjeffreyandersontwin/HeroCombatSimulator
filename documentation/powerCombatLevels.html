Power Overview

Power Combat Levels

Description

This class implements the skill Combat Levels.  The Combat Levels can be configured to work whenever the character uses a specific set of abilities, each of which has to be added to the "active" abilities, or it can be configured to always be active.

Functional Overview

Combat Levels works by adding an effect called Combat Level Watcher to the source character.  The watch sits and looks for any ability in it's "active" list to be activated (via the abilityIsActivating() effect method).  When it sees that ability, it adds a effect CombatModifier to the source, which lasts until the characters next phase.
This power is mark as persistent, since it needs to be activated during all combat, even if the combat modifiers are currently applied.
This power adds an action/menu item to allow for reconfiguration of the combat levels.  Currently, the combat levels are all applied to the OCV of the source when initially configured.

Known Caveats

None.
