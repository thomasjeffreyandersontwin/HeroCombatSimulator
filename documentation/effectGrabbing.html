Effect Overview

Effect Grabbing

Description

Applied to characters which are currently grabbing another character.

Functional Overview

The effectGrabbing is applied to the character by the effectGrabbed.  The addEffect method checks for a casual strength escape using the maneuverGrab.challengeGrab method.
While the effectGrabbing is applied, the addActions method will add a Squeeze, Throw, and Release actions to the special list.

Known Caveats

None.
