Power Overview

Dodge

Description

This power implements the martial maneuver dodge.  Although this is considered a maneuver within champions, it is not a maneuver according to HCS terminology.

Functional Overview

Dodge is implemented by creating an empty power and setting the Ability.DCVBONUS value/pair for DCV adjustment.

Known Caveats

None
