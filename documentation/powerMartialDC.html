Power Overview

Skill Martial DCs

Description

This class implements the skill Martial Damage Class Levels.  The Martial DCs work with a specific set of abilities, each of which has to be added to the "active" abilities.  The number of DCs can also be configured.

Functional Overview

Martial DC works by adding an effectMartialDC to the source.  effectMartialDC watches for one of the "active" abilities to be activated, and then adds the Martial.DC value/pair to the battleEvent.

Known Caveats

None.
