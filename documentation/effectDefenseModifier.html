Effect Overview

Effect DefenseModifier

Description

Implements an effect when modifiers the defenses (PD, rPD, ED, rED) of the character.  Used by Armor and Force Field to apply modifiers.

Functional Overview

Uses the addDefenseModifiers method to apply defenses to the character.
The defensive values are static, so the effect must be removed and reapplied to change values.

Known Caveats

None.
