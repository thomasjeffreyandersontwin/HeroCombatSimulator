Power Overview

Power Growth

Description

Implements the power Growth.  This power adds level increments of +5 STR, -1" knockback to the character, +1 BODY and +1 STUN.  For every 3 levels -2 DCV is added to the character and if the remainder of levels is at a half way point (ex: 2 levels left) he would receive an additional -1 DCV.

Functional Overview

When triggered, the power generates a custom effect called effectGrowth, with 3 subeffects of type aid to increase the specified stats based on the above increments and then custom code similar to effectKnockBack modifies the target's knockback resistance.  Finally addDCVDefenseModifiers borrowed from effectCombatModifiers is used to create a DCV penalty based on increments of 3 levels.

Known Caveats

        powerGrowth does not track mass, +2 to others PER rolls, height, width or reach at this time.
