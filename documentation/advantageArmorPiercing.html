Advantage Overview

Advantage Armor Piercing

Description

Implements Armor Piercing.  The number of levels of Armor Piercing is configurable.

Functional Overview

Uses prepower method to cut the PD, rPD, ED, and rED of target in half.  Before cutting the defenses, it check if there is an index Defense[].NAME equal to hardened, and if so, it check on Defense[].LEVELS for that index to make sure there are more Armor Piercing levels then hardened levels.

Known Caveats

Hardened Defenses are not properly used.  Currently having 1 point of hardened defenses allows all defense to be hardened to that level.  If multiple defense exist, only the first one will be considered, and that may not be the best one.
