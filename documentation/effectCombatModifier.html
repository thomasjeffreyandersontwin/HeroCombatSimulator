Effect Overview

Effect CombatModifier

Description

Implements a generic modifier to the OCV/DCV of the target.  The modifier is persistent, but only last until the beginning of the characters next phase.
This effect is used by all abilities to apply the "built-in" OCV/DCV modifiers for abilities and maneuvers.

Functional Overview

Uses addDCVDefenseModifiers and addOCVAttackModifiers to apply OCV/DCV modifiers to character.
Returns true from the prephase method, causing the effect to be removed at the beginning of the characters phase.
The OCV/DCV values are static and can't be changed once set.

Known Caveats

None.
