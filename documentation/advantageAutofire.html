Advantage Overview

Advantage Autofire

Description

Implements advantage Autofire.  The maximum number of shots of autofire is configurable.

Functional Overview

Uses the Ability.ISAUTOFIRE value/pair to tell the battleEngine to treat ability as an autofire ability and process accordingly.  The Ability.MAXSHOTS value/pair is set to indicate the number of shots which can be fired.

Known Caveats

None.
Advantage Information
Affects DC TRUE
Priority 2
Parsed Parameters
MaxShots Integer Ability.MAXSHOTS
        Indicates the maximum number of shot which can be taken at one time.
Other Value/Pairs
IsAutofire TRUE Ability.ISAUTOFIRE
        Indicates to BattleEngine that this is an autofire ability.
