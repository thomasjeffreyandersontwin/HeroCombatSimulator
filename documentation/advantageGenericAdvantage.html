Advantage Overview

Advantage Generic Advantage

Description

Implements an advantage that does not exist in the 4th edition rules.  The point of this advantage is to create a placeholder for advantages that are not complete or have no HCS relevancy BUT still allows for CPs to be computed correctly for the power.  In the future this advantage could be used as a building block location for GMs to make their own advantages or to create a modified copy of an existing advantage.

Functional Overview

This advantage creates the option to create an advantage where the user inputs the description and assigns the multiplier from 1/4 -3.

Known Caveats

None known.
