Advantage Overview

Advantage Reduced Endurance

Description

Implements Reduced Endurance.  Currently, only the level of reduction (1/2 or 0 END) is configurable.

Functional Overview

Currently, the presence of the advantage Reduced Endurance is explicitly checked when the BattleEngine calculates END use.

Known Caveats

None.
