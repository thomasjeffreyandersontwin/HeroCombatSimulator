Effect Overview

Effect Grabbed

Description

Indicates that the character has been grabbed by someone.  The grabbed effect works along with the grabbing effect.  Effect grabbed is applied by the grab power.

Functional Overview

When applied to a character, the addEffect method of grabbed automatically creates the effectGrabbing and applies it to the source of the grab.  If the grabbed effect is removed, effectGrabbing is automatically removed also.
The addDCVDefenseModifiers method is used to reduce the DCV of the grabbed character by one half.
The addAction method provides an "escape" ability.

Known Caveats

None.
