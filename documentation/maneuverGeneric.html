Power Overview

Generic Attack Maneuver

Description

The power implements a generic attack maneuver.  It can be setup to be a normal or killing attack with an arbitrary number of damage classes.

Functional Overview

This power uses the BattleEvent's generic damage generation to generate appropriate damage.

Known Caveats

None.
