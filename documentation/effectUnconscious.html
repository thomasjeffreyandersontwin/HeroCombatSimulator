Effect Overview

Effect Unconscious

Description

Indicates the character is currently unconscious.  Constant powers are deactivated at the end of segment when a character is unconscious.

Functional Overview

All constant powers are deactivated at the end of segment when the effect is added.  This is done via the addEffect method.  For each activated, constantability belonging to the character, a delayed deactivation event in created and posted.
When the effect is remove, the removeEffect method sets the END of the character equals to the current Stun of the character.
The addDCVDefenseModifiers method is used to set the DCV of the character to 0 normal.
During the postdefense method, all stun applied to the character is doubled.
Special BattleEngine/GUI code disable all powers except pass and recovery.
