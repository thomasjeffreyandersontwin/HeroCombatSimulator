Effect Overview

Effect Shrinking

Description

Implements Shrinking.  Used by power Shrinking.

Functional Overview

  • Uses predefense to add Knockback.MASS to the battleEvent.  The battleEngine handles the MASS processing beyond that The Knockback.MASS parameter is set when the effect is created.
  • Uses the addDCVDefenseModifiers  method to apply a DCV bonus to the character. 

Known Caveats

Does not track mass, +2 to others PER rolls, height, width or reach at this time.
