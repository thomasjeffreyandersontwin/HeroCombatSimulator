Limitation Overview

Limitation Extra Time

Description

Implements extra time.  The amount of extra time ("1 Phase", "1 Turn", "1 Minute", "5 Minutes", "1 Hour", "5 Hours") is configurable.

Functional Overview

Uses the build in delayed activation value/pairs to delay the activation of the ability appropriately.

Known Caveats

None.
Limitation Information
Class champions.powers.limitationExtraTime
Priority 2
Parsed Parameters
ExtraTimeLevel String ExtraTime.LEVEL
        Indicates the amount of time necessary to activate the ability, one of "1 Phase", "1 Turn", "1 Minute", "5 Minutes", "1 Hour", "5 Hours".
