Power Overview

Power Clinging

Description

Implements Clinging power.  Allows the source to cling to surfaces and run on them as if the surface was level.

Functional Overview

When power is triggered, power generates effectClinging.

Known Caveats

The following features are not implemented:
1) when someone tries to pull the source off the surface it should require a normal STR vs. clinging STR roll. For complete implementation there should also be a way to test the STR vs. STR against the surface so the surface could actually break.
2) There also should be a -1D6 to the Knockback rolls against a target with clinging.
3) The description in the character editor for clinging does not update correctly when the strength is changed.
.
