Power Overview

Maneuver Haymaker

Description

Implements the standard haymaker maneuver.  Haymaker requires extra time to activate, but does an additional 50% damage.

Functional Overview

Haymaker uses the BattleEngine automatic damage generation.  However, it uses adjustDice() method to add additional maneuver DCs based on the amount of strength the target has.  It also set various activation delay parameters to delay the attack.

Known Caveats

When using the haymaker attack with a killing power or if using the haymaker in a heroic campaign with DCs limited by the base attack, an incorrect amount of strength may be calculated for the attack.  This occurs due when haymaker adds additional maneuver DCs past the attacks maximum, and the BattleEngine compensates by reducing the strength used.  Typically this leads to slightly inaccurate END costs, but final DCs are correct.
