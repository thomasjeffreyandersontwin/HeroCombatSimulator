Effect Overview

Effect KnockbackResistance

Description

Implements Knockback resistance.  Used by power knockback resistance.

Functional Overview

Uses predefense to add Knockback.RESISTANCE to the battleEvent.  The battleEngine handles the resistance processing beyond that.
The Knockback.RESISTANCE parameter is set when the effect is created.  If the ability changes, the effect must be recreated.

Known Caveats

None.
