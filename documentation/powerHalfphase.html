Power Overview

Generic Halfphase action

Description

Implements a generic halfphase action.  When triggered, half of the sources phase is used.  Other then time being advanced, no other effect occur.

Functional Overview

The power is marked as a Half phase action.  The battleEngine automatically advances the source phase correctly.  No effects occur.
Autosource is set to true, so this power can always be applied against the currently active character.

Known Caveats

None.
