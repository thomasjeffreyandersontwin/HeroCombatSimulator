Power Overview

Power Flash Defense

Description

Implements Flash defense.  Since senses are not implemented by HCS, the sense protected is configured as an arbitrary string.

Functional Overview

Creates effectFlashDefense for source.  effectFlash looks for flash defense when applied to target.

Known Caveats

None.
