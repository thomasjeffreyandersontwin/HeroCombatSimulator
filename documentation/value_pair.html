Object Key Type Value Description
Ability Ability.AUTOSOURCE Boolean TRUE,FALSE If TRUE BattleEvents generated from the ability will have their source set to the current active target.  If FALSE, Ability.SOURCE must be set to the appropriate Target.
Ability Ability.SOURCE Target Target Source used for BattleEvents if Ability.AUTOSOURCE == FALSE.
Target Target.DELAYEDATTACK Boolean TRUE,FALSE If TRUE the target is currently in the process of making a delayed attack and cannot take another action until the delayed attack completes.
Ability Target[] Index * List of targets targeted by an ability during an attack.  A minimum of 1 target must be specifies for any completed attack.
Ability Target[].TARGET Target Target Instance of target class.
Ability Target[].HITROLL Integer * Attack Roll made during BattleEngine attack processing.
Ability Target[].ROLLNEEDED Integer * Attack roll necessary for source to hit target during attack processing.
Ability Target[].HIT Boolean TRUE,FALSE Indicates wheather the target was sucessfully hit by the attack.
Ability Ability.ISAUTOFIRE Boolean TRUE,FALSE If True, indicates the attack is an autofire attack.  Note: This is an ability value/pair, so it is set outside the attack processing.
Ability Ability.ISAE Boolean TRUE,FALSE If True, indicates attack is an area effect attack.  Note: This is an ability value/pair, so it is set outside the attack processing loop, typically by the advantage AE.
BattleEvent Attack.ISSPREAD Boolean TRUE,FALSE If True, indicates attack should be spread.  Should only be set when ability is EnergyBlast.  Not useable with Area Effect.  Note: This is a battleEvent value/pair and is typically set by the attack inline panel.
BattleEvent Attack.ISMOVEBY Boolean TRUE,FALSE If Ture, indicates attack should be a moveby.  Only set when ability or maneuver is a moveby.  Not compatible with Autofire attacks or Area Effect.  Note: This is a battleEvent value/pair and is typically set by the attack inline panel.
Ability Abiilty.NAME String * Name of the ability.
Ability Power.POWER Power Power Object of type power.
Ability Power.NAME Stirng * Name of Power.
BattleEvent Normal.STR Integer Integer Strength of attacking target.  Initially set to strength of target.  Modified to the maximum amount of usable strength for that ability/maneuver combination by calculatedDCs.  Used to determine STR based END use.
Target Effect[].EFFECT Effect Effect Instance of type effect.  Represents the effects currently applied to the target.
Ability Ability.DESCRIPTION String * Description of the ability.  Used by ability.getLongDescription() to generate ability description text.  Contains only the Power description information.
Ability Ability.SPECIALEFFECT String * Special Effect of power. Used by ability.getLongDescription() to generate ability description text.
Ability Advantage[].NAME String * Name of advantage
Ability Advantage[].LEVELS Integer * Number of levels configured for advantage.  Use of this value/pair is deprecated in most advantages.
Ability Advantage[].DESCRIPTION String * Description of advantage applied to ability.  Used by ability.getLongDescription() to generate ability description text.
Ability Advantage[].ADVANTAGE Advantage * Instance of Advantage.  Must be non-null.
Ability Advantage[].PRIORITY Integer 0 to 5 Priority of advantage when applying advantage effects.  See BattleEvent documentation for additional information. Defaults to 2.
Ability Limitation[].NAME String * Name of Limitation
Ability Limitation[].LEVELS Integer * Number of levels configured for limitation.  Use of this value/pair is deprecated in most limitation.
Ability Limitation[].DESCRIPTION String * Description of limitation applied to ability. Used by ability.getLongDescrption() to generate ability description text.
Ability Limitation[].LIMITATION Limitation * Instance of Limitation.  Must be non-null.
Ability Limitation[].PRIORITY Integer 0 to 5 Priority of limitation when applying limitation effects.  See BattleEvent documentaiotn for additional information. Defaults to 2.
Ability Ability.ISRECOVERY Boolean TRUE,FALSE Parameter to control whether ability is usable during post-12 phase.  If Ability.ISRECOVERY is FALSE, the ability will be disabled during post-12.
Ability Ability.TIME String FULLMOVE, ATTACK, HALFMOVE, INSTANT Parameter which determines amount of time necessary to activate ability.
FULLMOVE: requires complete phase
ATTACK: requires remainder of phase to complete.
HALFMOVE: requires half a phase to complete.
INSTANT: completes instantly.
Ability Ability.ACTIVATED Boolean TRUE,FALSE Determines whether the ability is currently activated.  Used in the battleEngine to determine whether an ability should be activated or deactivated when the user triggers an ability.
Ability Ability.PTYPE String PERSISTENT, CONSTANT, 
INSTANT		 Determines how, when, duration of ability activation.  Currently incorrectly implemented.  PTYPE currently has the following effects:
PERSISTENT: Power which remains on when target is unconscious.  Persistent powers are currently automatically triggered when a target becomes active.
CONSTANT: Power which remains on once triggered, until it is either turned off or target goes unconscious.
INSTANT: Power which is activated only momentarily before being immediately switched off.
Ability Ability.OCVBONUS Integer * OCV combat modifier applied to source target when ability is triggered.  May be positive or negative. May exist on both normal abilities and maneuvers.
Ability Ability.OCVBONUS Integer * DCV combat modifier applied to source target when ability is triggered. May be positive or negative.  May exist on both normal abilities and maneuvers.
Ability Ability.FIXEDCP Integer >=0 Character Point value used for ability if Ability.FIXEDCPENABLED == TRUE.  When enabled, controls appropriate calculations, such as END.
Ability Ability.FIXEDCPENABLED Boolean TRUE,FALSE Enables Ability.FIXEDCP for character point calculations.
Ability Ability.ATYPE String Attack type
Ability Die[].DIENAME String * Name of Die necessary to complete attack.  Corresponding <DIENAME>... information must exist.
Ability <DIENAME>.DIE String D6 notation Size of Die in D6 notation.  DIENAME is an string indexed by Die[].DIENAME.
Ability <DIENAME>.DIEDESC String * Description of die displayed when requesting die rolls. DIENAME is an string indexed by Die[].DIENAME.
Ability Ability.MTYPE String MELEE, RANGED Parameter required for attack abilities.  Indicates type of attack, either Melee or Ranged.  Used to determine if strength can be used to add damage to attack.
Ability Ability.KTYPE String NORMAL, KILLING, SPECIAL Parameter required for attack abilities.  Indicates type of damage, either Normal or killing.  Used to determine type of damage generated by the battleEngine for a normal attack.
Ability Ability.ISRANGED Boolean TRUE,FALSE Parameter specifying whether attack can hit ranged targets.  If ISRANGED is TRUE, the range OCV modify will be added to the OCV/DCV modifier dialog.  Not all ranged attacks require ISRANGED, but it is set by default to true for RANGED MTYPE attacks.
Ability Ability. CANUSERANGEDMANEUVER Boolean TRUE,FALSE Parameter specifying whether attack can use Ranged maneuvers.  If TRUE, the maneuvers menu will contain the appropriate ranged maneuvers for this ability.  Set to TRUE by default for RANGED MTYPE attacks.
Ability Ability. CANUSEMELEEMANEUVER Boolean TRUE,FALSE Parameter specifying whtehr attack can use Melee maneuvers.  If TRUE, the maneuvers menu will contain the appropriate melee maneuvers for thsi ability. Set to TRUE by default for MELEE MTYPE attacks.
Ability Ability.TARGETSELF Boolean TRUE,FALSE If set to true, attacks automatically target the source of the ability.
Ability Ability.ADVCOST Double * Cost of all the advantages applied to ability.
Ability Abiltiy.ADVDCCOST Double * Cost of DC effecting advantages.  Used to calculated D6 modifiers.
Ability Ability.LIMCOST Double * Cost of all the limitation applied to ability.
Ability ReducedEndurance.LEVEL String "1/2 END","0 END" Level of reduced endurance advantage applied to power.  Used to calculate the endurance cost of activating or maintaining an ability.
Ability Ability. DELAYEDACTIVATE Boolean TRUE,FALSE Indicates that activation of this ability takes extra time.  BattleEngine uses this to determine if a two stage activation is necessary.
Ability Ability.DELAYEXCLUSIVE Boolean TRUE,FALSE If the ability is a delayed activation ability, DELAYEXCLUSIVE specifies that the target can not perform any other actions until the ability finishes activating.  All attacks which require targeting rolls should be delay exclusive.
Ability Ability.DELAYCOUNT Integer * Specifies the number of periods necessary to delay before the ability finishes activating.  Periods length is defined by DELAYTIME.
Ability Ability.DELAYTIME Integer 0 to 6 Specifies the delay period in DELAYEDACTIVATE ability.  Period length = 
0) segment
1) phase
2) turn
3) minute
4) 5 minutes
5) 1 hour
6) 5 hours
Ability Ability.DELAYTILLLAST Boolean TRUE,FALSE IF DELAYTILLLAST is TRUE, a delayed activation ability will be the last ability activated during the appropriate segment.  If it is false, the ability will be activated according to the activating target's dexerity.
Ability Ability.DELAYINTERRUPTLEVEL Integer ... ...
Ability Ability.DELAYDCVMOD Double * If the ability is a delayed activation ability, the DCV of the activating target is multiplied by the DELAYDCVMOD value.  Default multiplier is 1.
Ability Ability.DELAYREASON String * Reason for the activation delay.  Used in effectDelayed description.
Ability Ability.CTYPE String POWER, SKILL, TELENT Type of ability.
Ability Ability.ISMOVEMENT Boolean TRUE,FALSE Ability is based on a movement power.  Movement powers are currently treated differently then normal powers by the battleEngine.
Ability Ability.ISMELEEMANEUVER Boolean  TRUE,FALSE Indicates whether an ability is a melee maneuver.  Melee maneuvers can be used by any ability which has the CANUSEMELEEMANEUVER v/p set to True.
Ability Ability.ISRANGEDMANEUVER Boolean TRUE,FALSE Indicates whether an ability is a ranged maneuver.  Ranged maneuvers can be used by any ability which has the CANUSERANGEDMANEUVER v/p set to True.
Ability Power.DEFENSE String PD,ED Indicates the type of defense to be applied agaist this ability during an attack.
Ability Ability.ISAUTOHIT Boolean TRUE,FALSE Indicates whether this ability always hits.
BattleEvent Knockback[] Index * Index with information necessary to determine knockback done by attack.
BattleEvent Knockback[].TARGET Target * Indicates which target knockback will be applied to.
BattleEvent Knockback[].AMOUNT Integer >=0 Indicates the number of inches of knockback to be applied.
Ability KnockbackRoll[] Index * Index with Knockback roll modification parameters.
Ability KnockbackRoll[].TYPE  String MOD Indexed of KB modifiers for ability.  MOD is a straight modifier.
Ability KnockbackRoll[].
DESCRIPTION		 String * Description of KB modifier.
Ability KnockbackRoll[].VALUE Integer * Amount of modification to the knockback roll.  Depending on the value of type, value may indicate different things.
Ability KnockbackRoll[].ACTIVE Boolean TRUE,FALSE Indicates this KB modifier is currently active.  By default, all knockback modifiers are active.
BattleEvent Attack.CURRENTSHOT Integer * Indicates the current shot of a series of autofire shots.  Should only be set by battleEngine.
BattleEvent Attack.ISSPRAY Boolean TRUE,FALSE If set to True, indicates the attack is an autofire spray attack.
BattleEvent OCVModifier[] Index * Index containing OCV modifiers applied during an attack.
BattleEvent OCVModifier[].NAME String * Name of OCV modifier.
BattleEvent OCVModifier[].DESCRIPTION String * Description of OCV modifier.  Displayed in OCV modifier dialog.
BattleEvent OCVModifier[].VALUE Integer * Amount to modify OCV by.
BattleEvent Attack.SPRAYWIDTH Integer >=1 Number of Hexes a Sprayed AF attack is spread across.  Set by attack dialog.
BattleEvent Attack.SHOTS Integer >=1 Number of shots to shoot in AF attack.  Set by attack dialog.
BattleEvent Attack.SPREADWIDTH Integer >=1 Number of Hexes an energy blast spreads against when it is spread.
BattleEvent Attack.ISSPREADM Boolean TRUE,FALSE Indicates whether an EB spread attack is a multi-target attack or a single target attack. 
BattleEvent Attacl.MOVEBYTARGETS Integer >=1 Indicates the number of targets being attacked in a multiple moveby.  Set by Attack Inline panel.
Ability Ability.ISSELECTIVEAE Boolean TRUE,FALSE Indicates an Area Attack ability is SELECTIVE.  Set by AE advantage.  When set, processAE makes a seperate to hit attack against each target in the AE area.
Ability Ability.ISNONSELECTIVEAE Boolean TRUE,FALSE Indicates an Area Attack ability is Non-Selective.  Set by AE advantage.  When set, processAE makes a seperate to hit attack against each target in the AE area.
Effectlist Effect[] Index * Index added to effectlist.  Contain informaiton about different effects to be applied to target or source of an attack.
Effectlist Effect[].EFFECT Effect * Effect Object to be applied by power.
EffectList Effect[].TTYPE String TARGET, SOURCE Indicates where an effect generated by an ability needs to be applied.  TARGET indicates apply effect to target.  SOURCE idicates apply effect to source.  Defaults to TARGET if blank.
Effect Subeffect[] Index * Index containing information regarding subeffects to be applied to target of effect.  Every Effect can contain subeffect to be applied to target, however often custom effect do not define any subeffects.
Effect Subeffect[].VERSUS String/ Ability * Indicates what is affected by subeffect.  Type depends on VERSUSTYPE of subeffect.  If VERSUSTYPE == STAT or STATCP, then VERSUS must be a string indicating the appropriate Stat such as BODY or CON.  If VERSUSTYPE == ABILITY, the VERSUS must be a Ability object.
Effect Subeffect[].NAME String * Name of Subeffect.  Typically used only for lookup purposes.
Effect Subeffect[].EFFECTTYPE String DAMAGE, DRAIN, AID, LIMITED_AID, LIMITED_DRAIN, SET Indicates how the subeffect is applied to VERSUS.
DAMAGE: Applies as unlimited reducation to appropriate value.
DRAIN: Applied as unlimited reduction to appropriate value.
AID: Applied as unlimited addition to appropriate value.
LIMITED_DRAIN: Applied as reduction to appropriate value, limited to minimum of 0.
LIMITED_AID: Applied as addition to appropriate value, limited to value maximum.
SET: Changes appropriate value to subeffect value.
Effect Subeffect[].DEFTYPE String PD,ED,NONE Indicates the type of defense to be used when applying the subeffect to a target.  Defaults to NONE.
Effect Subeffect[].DEFSPECIAL String NORMAL, NONE, NND, AVLD, KILLING Indicates special methods of applying defenses to subeffect. 
NORMAL: apply defenses as usual.
NONE: do not apply any defense.
NND: treat attack as No Normal defense attack.
AVLD: treat attack as Attack versus limited defense.
KILLING: treat attack as killing attack.
Effect Subeffect[].VERSUSTYPE String STAT,STATCP, ABILITY Indicates what subeffect is applied to and what value is measured in.
STAT: apply to stat and value is in stat points.
STATCP: apply to stat and value is in character points.
ABILITY: apply to ability and valueis in character points.
Effect Subeffect[].VALUE Integer * Amount to apply.  Metric indicated according to VERSUSTYPE.
Ability Ability.ISMOVETHROUGH Boolean TRUE, FALSE Indicates ability is a movethrough.  If either the ability or maneuver of an attack is a movethrough, special movethrough attack processing will occur.
Effect PD.DEFENSE Integer >=0 Amount of PD Defense target of effect has.
Effect ED.DEFENSE Integer >=0 Amount of ED Defense target of effect has.
Effect rPD.DEFENSE Integer >=0 Amount of rPD Defense target of effect has.
Effect rED.DEFENSE Integer >=0 Amount of rED Defense target of effect has.
Effect NND.DEFENSE String * Description of defense which defeats no normal defense attack.  Necessary for an effect with NND marked subeffects.
BattleEvent BattleEvent.DCCALCULATED Boolean TRUE,FALSE Indicates the DC calculations for the battleEvent have been made.
BattleEvent Ability.TEMPTIME String FULLMOVE, ATTACK, HALFMOVE, INSTANT Time required for attack/activation to complete.  Overrides Ability.TIME from Ability.  Values mean the same as Ability.TIME.
