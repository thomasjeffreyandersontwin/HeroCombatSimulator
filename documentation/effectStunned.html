Effect Overview

Effect Stunned

Description

When applied to character, effectStunned indicates that the character is stunned.  Constant powers are deactivated at the end of segment when a character is stunned.

Functional Overview

All constant powers are deactivated at the end of segment when the effect is added.  This is done via the addEffect method.  For each activated, constant ability belonging to the character, a delayed deactivation event in created and posted.
The addDCVDefenseModifiers method is used to set the DCV of the character to 1/2 normal.

Known Caveats

None.
