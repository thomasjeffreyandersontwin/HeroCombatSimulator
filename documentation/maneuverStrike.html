Power Overview

Maneuver Strike

Description

Implements a generic strike maneuver.  This maneuver is almost identical to the generic maneuver.  The damage class and type of damage (normal or killing) can be configured to produce an standard martial strike.

Functional Overview

Uses the standard BattleEngine damage generation.  DC and Type of attack are setup during configuration to reflect configuration of maneuver.

Known Caveats

None.
