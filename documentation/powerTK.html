Power Overview

Power Telekinesis

Description

Implements the Telekinesis Power.  Amount of strength the TK can apply is configurable.

Functional Overview

TK is implemented as a standard Hand-To-Hand attack power, which happens to have ranged abilities.  It uses a Strength type of RANGED, so that normal strength is not applied.  TK Strength is applied as Base DC for the attack, not as actual strength, and will not automatically be adjusted by the BattleEngine when too much TK stength is use (ie. beyond the max strength for a particular attack).
TK is setup to be able to use Melee maneuvers and to not be able to use Ranged maneuvers.

Known Caveats

Strength is not applied as normal strength, but rather as Base.DCs.
TK Strength is not adjusted by BattleEngine with regards to max strength for an attack.
Although TK Strength can be used with standard Melee maneuvers, it currently can't be used to pick up a weapon, and strike an opponent at a distance.
