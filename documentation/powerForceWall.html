Power Overview

Power Force Wall

Description

Implements Power Force Wall.  The PD and ED of the ForceWall are configurable.

Functional Overview

When triggered, a effectForceWall is created and applied to the source.  The effectForce creates a force wall target and links the ability to the target.  When the target is destroyed, the ability is deactivated, and vice versa.

Known Caveats

Currently, the ability time is marked as instant.  However, since an attack roll is required, it should probably be marked as time attack.  The GM must advance time as necessary.
