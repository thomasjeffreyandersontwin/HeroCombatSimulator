Advantage Overview

Advantage No Normal Defense

Description

Implements No Normal Defense.  Currently whenever an ability has the NND advantage, the BattleEngine prompts to determine if the NND worked or not.  The Attack Type and Defense for the NND is configurable.

Functional Overview

Uses the prepower method to watch all effect generated by power.  If the effect includes any type of subeffect, NND will set the Subeffect[].DEFSPECIAL to NND.  The battleEngine will then treat the effect as an NND attack.

Known Caveats

Not compatible with anything that uses the DEFSPECIAL value/pair, including Killing attacks and AVLD.
Currently, NND will stop knockback from occuring, but it does so through a hack.  This needs to be fixed.
