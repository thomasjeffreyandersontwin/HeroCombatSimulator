Effect Overview

Effect CombatLevel

Description

Implements effect which automatically applied OCV, DCV, and DC modifications when appropraite.  Used with the Combat Level skill.

Functional Overview

Uses abilityIsActivating method to watch for ability/maneuver activation.  When an appropriate ability is activated (as determined by the CanUseCL[] index attached to the Combat Level skill), effectCombatLevel creates and adds an effectCombatModifier with the appropriate OCV/DCV modifications to the source.
Uses adjustDice method to watch for ability/maneuver activation.  When an appropriate ability is activated (as determined by the CanUseCL[] index attached to the Combat Level skill), effectCombatLevel adds to the battleEvents Combat.DC value/pair.
The effectCombatLevel is linked to an ability and values are dynmically implemented.  It expects the ability to have CanUseCL[] index, Power.OVERALL, CombatLevel.DCBONUS, CombatLevel.OCVBONUS, and CombatLevel.DCBONUS value/pairs.

Known Caveats

In the case of an overall level, that level is still only triggered when an ability is activated.  Situations may exist where the character does not get the benifit of the level even though it should be activated.
