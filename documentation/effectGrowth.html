Effect Overview

Effect Growth

Description

Implements Growth.  Used by power Growth.

Functional Overview

  • Uses predefense to add Knockback.RESISTANCE to the battleEvent.  The battleEngine handles the resistance processing beyond that The Knockback.RESISTANCE parameter is set when the effect is created.  If the ability changes, the effect must be recreated.
  • Uses the addDCVDefenseModifiers  method to apply a DCV penaltyto the character.  The DCV penalty is static, so the effect must be removed and reapplied to change values.
  • Uses 3 subeffects of type  AID to add STR, BODY and STUN to the character.

Known Caveats

Does not track mass, +2 to others PER rolls, height, width or reach at this time.
