Advantage Overview

Advantage Area Effect

Description

Implements Area Effect advantage.  Currently, the only configurable option is the AE type: Normal, Selective, and Non-Selective.

Functional Overview

AE works by marking the Ability.ISAE value/pair to TRUE.  The battleEngine takes care of the rest.  The Ability.ISSELECTIVEAE and Ability.ISNONSELECTIVEAE are set appropriately for the AE type.

Known Caveats

None.
Advantage Information
Class champions.powers.advantageAreaEffect
Affects DC FALSE
Priority 2
Parsed Parameters
Type String AE.TYPE
        Indicates the type of AE it is, one of Normal, Selective, or Non-Selective.  The Ability.ISSELECTIVEAE and Ability.ISNONSELECTIVEAE are set appropriately for the AE type.
Other Value/Pairs
IsAE TRUE Ability.ISAE
        Indicates to the BattleEngine that the Area Effect attack processing should occur.
