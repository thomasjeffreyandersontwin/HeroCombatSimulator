Limitation Overview

Limitation Concentration

Description

Implements Concentration.  The level of concentration (1/2 DCV or 0 DCV) is configurable.

Functional Overview

Uses the build in delayed activation value/pairs to delay the activation of the ability appropriately.  The effectDelayed takes care of the DCV modifications.

Known Caveats

None
Limitation Information
Class champions.powers.limitationConcentration
Priority 2
Parsed Parameters
ConcentrationLevel String Concentration.LEVEL
        Indicates the level of concentration, either "1/2 DCV" or "0 DCV".
