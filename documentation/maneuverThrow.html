Power Overview

Maneuver Throw

Description

Implements the standard throw maneuver.   The maneuver can be configured with additional DCs to simulate martial throws which cause additional damage.

Functional Overview

Uses the standard BattleEngine damage generation.  However, the maneuver also generates it's own knock down effect to apply to the target.  The preactivation() method is used to present the user a GUI to input the velocity of the attacker.  AdjustDice() uses that velocity to determine final DC of the attack.

Known Caveats

None.
