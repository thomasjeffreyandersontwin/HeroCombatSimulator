Power Overview

Take Recovery

Description

Implements the standard recovery action.

Functional Overview

When triggered, the power generates a standard effect, which subeffect applying limited_aid to STUN and END (if conscious) to source.
The power is marked as autosource, so the currently active characters turn is alway passed.  The power is also marked as isRecovery, so that a character can pass their post-12 recovery if desired.
IsEnabled is overriden to restrict unconscious characters from taking recoveries when they are not eligible.
The appropriate delayed attack value/pairs are added to make the recovery action last until the end of segment.

Known Caveats

None.
