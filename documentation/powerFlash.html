Power Overview

Power Flash

Description

Implements the power flash.  Senses and sense groups are not implemented yet, so flash currently is configurable with just a description of the sense (normal sight) and whether the sense is a targeting sense.  The D6 of the attack is also configurable.

Functional Overview

Flash creates an effectFlashed, which reduces OCV/DCV of the target appropriately and degrades as phases pass.

Known Caveats

Currently no sense are implemented.
