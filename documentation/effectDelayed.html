Effect Overview

Effect Delayed

Description

Generic effect used whenever an ability has a delay during activation.  effectDelayed adjusts the DCV of the character and watches the character for anything that might disrupt the delay.

Functional Overview

Uses the posteffect method, which is called ever time a new ability is added to a character, to watch for effects which would cause the delay to be disrupted.  After it sees one of those events, it returns true from posteffect, triggering it's own removal.
The removeEffect method takes care of all cleanup responsibilities, while the addDCVDefenseModifiers method is used to apply defense modifiers.

Known Caveats

Doesn't seem to always work.

Parsed Parameters
INTERRUPTLEVEL Integer Effect.INTERRUPTLEVEL
        Specifies what effects will interrupt the delay:  0) Nothing, 1) Body/Stun damage or drain, 2) Body/Stun damage or drain and knockback, 3) any damage, drain, or knockback.
DCVMod Double Effect.DCVMOD
        Specifies the multiplier applied to the character.  1 indicates no DCV changes.  0 indicates the DCV should be set to 0.  Anything in between is treated as a percentage modification.
Event BattleEvent Effect.EVENT
        Specifies the event which will be triggered once the delay is over.
