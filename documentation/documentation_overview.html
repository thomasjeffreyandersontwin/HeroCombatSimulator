HCS Documentation Project

The HCS Documentation project (HCSD) contains all of the documentation necessary to develop in the HCS environment.  HCSD is broken up into multiple phases.  See Table below for information about the specific phases of the project and current status of each phase.  Whenever possible, HCSD will be done inline and javadoc will be used to automatically generate the final documentation.
 
 
Phase
Description
Status
Package & Class Overview Basic overview information for all major packages and classes used in HCS. Incomplete
Battle Engine Flow Overview of the flow of the BattleEngine, including BattleEvent dispatching. Incomplete
Battle Engine Detail Detailed documentation of every function in the Battle Engine class, including guidelines for using BattleEngine functions outside the main BattleEngine event loop. Incomplete
BattleEvent Detail Detail of BattleEvent types, functions, and undo guidelines. Incomplete
Battle Detail Detail info on the internal workings of Battle class and how to implement battles. Incomplete
Value-Pair Documentation Detailing of every value-pair usable in HCS and which do what.  Incomplete
Power/Advantage/Limitation Overview Informtion on how PAL code works and how to create powers, advantages, and limitations. Incomplete
PAL Detail Detailed explination of how each specific PAL works, caveats associated with each PAL, and BattleEngine modification necessary to enable PAL to function correctly. Incomplete
Effect Overview Overview of how effects work and what they do. Incomplete
