Power Overview

Power Entangle

Description

Implements the Entangle power.  Currently, this only applies an entangle effect to the target and creates the entangle target

Functional Overview

Generates an effectEntangle to apply to the target.  The effectEntangle does the rest, including creating a roster target to represent the entangle itself.

Known Caveats

Doesn't implement any special entangle properties.
Currently entangles to no block any damage applied directly to the target.
