Power Overview

Power Shrinking

Description

Implements the power Shrinking.  For every level, +2 DCV is added to the character as well as +3” to the distance a character travels when knockback is used.

Functional Overview

When triggered, the power generates a custom effect called effectShrinking, which adds 3 per level to the Knockback.MASS value pair.  This does not increase damage just distance.  Finally addDCVDefenseModifiers borrowed from effectCombatModifiers is used to create a DCV bonus of +2 per level.

Known Caveats

        powerShrinking does not track character mass, height, adjustment to PER rolls, or damage increased due to growth momentum.
