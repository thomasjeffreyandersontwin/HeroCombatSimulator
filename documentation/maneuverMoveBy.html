Power Overview

Manevuer Move By

Description

Implements standard Move By maneuver.  The maneuver can be configured with additional DCs to simulate martial move throughswhich cause additional damage.

Functional Overview

Uses the standard BattleEngine damage generation.  Preactivation() method is used to present a GUI to the user to determine the velocity of the source.  AdjustDice() and addOCVAttackModifiers() then adjust the DC and OCV of source according to that velocity.  The BattleEngine contains special code to handle multiple move bys correctly.

Known Caveats

Multiple moveby's can not be used in combination with other special attack types such as autofire or area effect.
